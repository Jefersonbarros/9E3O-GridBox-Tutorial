# 9E3O-GridBox-Tutorial

ChimeraX commands and step-by-step guide for preparing PDB 9E3O and generating a docking grid box.

## Repository Structure

```text
9E3O-GridBox-Tutorial/
├── ChimeraX/
│   ├── commands.cxc
│   └── screenshots/
├── GridBox/
│   ├── grid_parameters.md
│   └── coordinates.txt
├── Redocking/
├── RMSD/
└── README.md
```

## Objective

This repository documents a reproducible workflow for:

- Preparing the PDB 9E3O structure in ChimeraX
- Identifying the binding site
- Generating a docking grid box
- Recording grid coordinates and parameters
- Performing redocking validation
- Calculating RMSD values

PDB ID: 9E3O
